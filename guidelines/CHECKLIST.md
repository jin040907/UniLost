## üìù Notes

### Before Uploading to GitHub

1. **Verify Database File Exclusion**
   - `unilost.db` is automatically excluded because it's included in `.gitignore`. ‚úÖ

2. **Environment Variable Settings**
   - The `SESSION_SECRET` environment variable must be set during deployment.
   - This is automatically generated by Render.

3. **Default Account Information**
   - The default accounts are for development purposes. Changing them in production is recommended.
   - **Student Accounts**: `student1` ~ `student10` / `1234`
   - **Administrator Accounts**: `admin1` ~ `admin10` / `admin123`

### Post-Upload Checklist

- [ ] Verify that all files have been uploaded to the GitHub repository
- [ ] Verify that the `.db` file has been excluded
- [ ] Verify that `node_modules` has been excluded

## üöÄ Deployment Method

### Step 1: Upload Your Code to GitHub

```bash
git init
git add .
git commit -m "Initial commit"
git remote add origin [your-repo-url]
git push -u origin main
```

### Step 2: Deploy from Render (Automatic)

- Access render.com
- Connect to GitHub repository
- Automatic deployment complete!
